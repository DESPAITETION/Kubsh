# kubsh - Custom Shell Implementation

A custom shell with VFS support for user information, developed as part of a university project.

## Features

- [x] Basic input/output with exit on Ctrl+D
- [x] Command history with file persistence
- [x] Environment variable management
- [x] External command execution
- [x] Signal handling (SIGHUP)
- [x] Virtual filesystem for user data
- [x] Disk information commands

## Quick Start

### Local Development

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/kubsh.git
cd kubsh

# Build and run
make build
./kubsh
```
